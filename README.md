# CryptoEstate – Coming Soon Landing Page

A modern **CryptoEstate Coming Soon** landing page built with **HTML, CSS, and JavaScript** for GitHub Pages.

## Overview

CryptoEstate is a blockchain-inspired real estate platform focused on secure digital property investment and ownership.

This landing page features:

* Luxury black & gold CryptoEstate branding
* Dynamic **14-day countdown timer**
* Responsive design for desktop and mobile
* Smooth glow and glassmorphism effects
* Lightweight and fast-loading HTML/CSS/JS setup

## Installation

Clone the repository:

```bash
git clone https://github.com/cyptoestate/cyptoestate.github.io.git
```

Open the project folder and make sure your logo image is named:

```bash
logo.png
```

Then open:

```bash
index.html
```

in your browser.

## GitHub Pages Deployment

1. Push files to your repository
2. Go to **Repository Settings**
3. Open **Pages**
4. Select the **main branch**
5. Save changes

Your website will be live at:

```bash
https://cyptoestate.github.io/
```

## Customization

### Change Countdown Duration

Inside `index.html`, locate:

```javascript
launchDate.setDate(launchDate.getDate() + 14);
```

Change **14** to any number of days.

### Replace Logo

Replace:

```bash
logo.png
```

with your own CryptoEstate logo while keeping the same filename.
